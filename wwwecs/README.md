# Pagina Next.js (Modern Stack)

## 1. Stack Tecnológica
*   **Framework:** Next.js 14+ (App Router)
*   **Linguagem:** TypeScript (Strict mode obrigatório)
*   **Estilização:** Tailwind CSS + `lucide-react` (ícones)
*   **Data Fetching:** Server Components + Next.js Route Handlers (BFF - Backend for Frontend)
